Deploy, run, and manage user applications installed on the reader.

| Command | Description |
|---|---|
| [get_user_apps](#op-get-user-apps) | List installed user applications |
| [install_user_app](#op-install-user-app) | Install a user application |
| [uninstall-user-app](#op-uninstall-user-app) | Uninstall a user application |
| [start_user_app](#op-start-user-app) | Start a user application |
| [stop_user_app](#op-stop-user-app) | Stop a running user application |
| [autostart_user_app](#op-autostart-user-app) | Configure a user app to start on boot |
| [set_req_usr_app](#op-set-req-usr-app) | Send a request to a user application |
| [set_dataToRG](#op-set-datatorg) | Forward data to the RFID gateway from a user app |
