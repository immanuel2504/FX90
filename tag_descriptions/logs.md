Retrieve, configure, and clear the reader's diagnostic and system logs.

| Command | Description |
|---|---|
| [get_logs](#op-get-logs) | Retrieve reader logs |
| [set_logs](#op-set-logs) | Configure logging settings |
| [get_logs_syslog](#op-get-logs-syslog) | Retrieve system (syslog) entries |
| [del_syslogs](#op-del-syslogs) | Delete stored syslog entries |
| [get_radio_pkt_logs](#op-get-radio-pkt-logs) | Retrieve radio packet logs |
| [del_radio_pkt_logs](#op-del-radio-pkt-logs) | Delete stored radio packet logs |
| [get_rg_error_logs](#op-get-rg-error-logs) | Retrieve RFID gateway error logs |
| [get_rg_warn_logs](#op-get-rg-warn-logs) | Retrieve RFID gateway warning logs |
| [get_rc_log](#op-get-rc-log) | Retrieve the reader control log |
