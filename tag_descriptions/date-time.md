Configure the reader's time zone and NTP time synchronization.

| Command | Description |
|---|---|
| [get_timeZone](#op-get-timezone) | Retrieve the configured time zone |
| [set_timeZone](#op-set-timezone) | Set the reader's time zone |
| [get_ntpServer](#op-get-ntpserver) | Retrieve the configured NTP server(s) |
| [set_ntpServer](#op-set-ntpserver) | Configure NTP server(s) for time synchronization |
