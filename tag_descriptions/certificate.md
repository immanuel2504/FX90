Manage the device and CA certificates used for secure connections.

| Command | Description |
|---|---|
| [get_certs](#op-get-certs) | List installed device certificates |
| [del_certs](#op-del-certs) | Delete an installed device certificate |
| [set_update_cert](#op-set-update-cert) | Install or update a device certificate |
| [refresh-cert](#op-refresh-cert) | Refresh or renew the device certificate |
| [set_installCACertificate](#op-set-installcacertificate) | Install a CA certificate |
| [get_CACertificates](#op-get-cacertificates) | List installed CA certificates |
| [del_CACertificate](#op-del-cacertificate) | Delete an installed CA certificate |
