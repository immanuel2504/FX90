Tag read data the reader streams as tags are observed, including mode-specific reports, directionality, zone transitions, and location history.

| Event | Description |
|---|---|
| [tagDataEvents](#op-tagdataevents) | Standard tag read data report |
| [mode_tag_data_events](#op-mode-tag-data-events) | Tag data report for the active operating mode |
| [directionality_tag_data_events](#op-directionality-tag-data-events) | Directionality (entry/exit) tag report |
| [zoneHistory](#op-zonehistory) | Tag zone transition history |
| [locationHistory](#op-locationhistory) | Tag location history (RTLS) |
