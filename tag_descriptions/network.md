Configure and inspect the reader's network connectivity, interfaces, hostname, and device identity.

| Command | Description |
|---|---|
| [get_network](#op-get-network) | Retrieve the current network configuration |
| [set_network](#op-set-network) | Apply network configuration (IP, DNS, and more) |
| [get_hostname](#op-get-hostname) | Retrieve the reader's hostname |
| [set_hostname](#op-set-hostname) | Set the reader's hostname |
| [get_availableWifiNetworks](#op-get-availablewifinetworks) | List Wi-Fi networks visible to the reader |
| [get_networkInterfaces](#op-get-networkinterfaces) | List available network interfaces |
| [get_readPoints](#op-get-readpoints) | Retrieve antenna read point configuration |
| [get_gpsCoordinates](#op-get-gpscoordinates) | Retrieve the reader's GPS coordinates |
| [get_eSimConfig](#op-get-esimconfig) | Retrieve eSIM/cellular configuration |
| [set_eSimConfig](#op-set-esimconfig) | Apply eSIM/cellular configuration |
| [set_nameAndDescription](#op-set-nameanddescription) | Set the reader's display name and description |
| [get_nameAndDescription](#op-get-nameanddescription) | Retrieve the reader's display name and description |
