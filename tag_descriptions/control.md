Control the RFID operating mode, start and stop inventory, and configure pre-selection filters.

| Command | Description |
|---|---|
| [get_mode](#op-get-mode) | Retrieve the current operating mode |
| [set_mode](#op-set-mode) | Configure the reader operating mode |
| [start](#op-start) | Start an RFID inventory/read operation |
| [stop](#op-stop) | Stop the active RFID inventory/read operation |
| [get_preSelection](#op-get-preselection) | Retrieve the pre-selection filter configuration |
| [set_preSelection](#op-set-preselection) | Configure pre-selection (Select) filters |
