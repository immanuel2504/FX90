The `PUT /cloud/gpo` REST endpoint updates GPO port state.

## 2. Endpoint Details

| Property | Value |
|---|---|
| REST Endpoint | `PUT /cloud/gpo` |
| Operation ID | `setGpo` |
| Authentication | Bearer token (`Authorization: Bearer <token>`) |
| Content-Type | `application/json` where a request body is required |
