# FXR90 AI Project Ideas

This folder contains the AI project ideas prepared from the FXR90 MQTT API documentation.

## Included Projects

| No. | Project | Purpose | PDF |
|---|---|---|---|
| 1 | Reader Overheating and Hardware Stress Predictor | Predict reader overheating, CPU/RAM stress, storage pressure, and stability risk. | [Open PDF](01_Reader_Overheating_Hardware_Stress/FXR90_Reader_Overheating_Hardware_Stress_Predictor.pdf) |
| 2 | Antenna and Cable Degradation Predictor | Predict antenna disconnects, cable degradation, RF path loss, and antenna misalignment. | [Open PDF](02_Antenna_Cable_Degradation/FXR90_Antenna_Cable_Degradation_Predictor.pdf) |
| 3 | Edge Maintenance Chatbot | No-cloud chatbot concept that runs as an FXR90 user app and explains local maintenance issues. | [Open PDF](03_Edge_Maintenance_Chatbot/FXR90_Edge_Maintenance_Chatbot.pdf) |

Each project folder includes:

- A presentation-ready PDF.
- An editable HTML source file.

## Recommended Build Order

1. Start with Antenna and Cable Degradation Predictor.
2. Add Reader Overheating and Hardware Stress Predictor.
3. Add Edge Maintenance Chatbot to explain the predictions locally without cloud dependency.
